**Repository:**

- Backend: [Nhom02_KTvTKPM_DHKTPM18A_Backend](https://github.com/HoangSonQS/Nhom02_KTvTKPM_DHKTPM18A_Backend)

**My Role:**

- Core backend developer
- Software architecture owner
- Defined and enforced architecture rules
- Reviewed and fixed commits that violated architecture boundaries
- Supported team members in backend implementation
- Handled deployment and CI/CD on server
- Took responsibilities similar to a technical lead in a 5-member team

**Key Features:**

- Modular monolith backend architecture
- Hexagonal / Clean Architecture inside modules
- RESTful API design
- Gateway configuration
- Rate limiting
- Load balancing across 2 backend instances
- PostgreSQL setup with 1 primary and 2 read replicas
- PostgreSQL streaming replication support
- Saga pattern for distributed business workflows
- Outbox pattern for reliable event handling
- Monitoring with Grafana, Loki, and Prometheus
- CI/CD deployment workflow
- Architecture rule enforcement and code review

**Technical Summary:**

- **Tech Stack:** Java, Spring Boot, PostgreSQL, Docker

- **Observability:** Prometheus, Grafana, Loki
